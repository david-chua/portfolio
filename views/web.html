Training For Gains



Inspired from using the gym and seeing people still using pen and pad to keep track of their progress, I decided to create a workout logging application that will keep track of a person's workout routine.

The Project was done with the following technologies: Ruby on Rails, PostgreSQL, Node.JS, Express.JS, AngularJS, and HTML/CSS, & Bootstrap

The Challenges

As this was my final project for General Assembly, it did prove to me my most challenging one. The project only had four days to be completed and it involved creating everything from the backend configuration with the models and their relations as well as creating the frontend with AngularJS. It was also the only project since Month 1 where I worked on a project individually.
While it was challenging, it did allow me to be more have a better time management and make decisions regarding the process or how my project would turn out. Having an Most Viable Product(MVP) checklist helped in that regards as I was able to cross out functionalities that I met and focus on finishing the rest of the project that I had remaining before going back and fixing those that I could with the time remaining
