AquaCycle: The First Sketch Design

This was my very first use of Sketch App in terms of creating prototypes. Prior to this, I would only create a hand drawn wireframe of an app and this would serve as my design before the actual creation of the UI interface. Using a prototype actually helped solidify my design process and helped me create a better UI.

Having the Sketch model allowed me to decide whether color themes would work together visually before I actually coded the colors. The use of Sketch also allowed me to have different iteration of the same pages which I could use for A/B testing.
